Digital Checkout System

Project Overview

Developed a Java program that implements an online checkout page. The program processes a catalog of products that the user selects to add to their cart. The system will manage inventory of the products added/removed to the cart and update the product catalog. Totals of products in the cart would then be calculated for the user to then checkout.
